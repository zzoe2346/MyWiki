- Singleton Pattern
- 생성자가 여러 번 호출되어도 실제로 생성되는 객체는 **하나**
- 많이 쓰임

```java
class Coin{
	private static Coin instance = new Coin();

	private Coin(){
	}

	public static Coin getInstance(){
		return instance;
	}
}
```

```java
class Main{
	public static void main(String[] args){
		Coin coin = Coin.getInstance();
	}
}
```
